## 🚀 Troubleshooting Common Issues

Problems happen occasionally. Use these solutions to solve the most common errors.

*   Game crashes on startup: Ensure that your graphics card drivers are current. Outdated drivers cause many conflicts with modded content.
*   Out of memory error: Change the memory allocation in your game launcher. Assign at least 6 GB of RAM to the game.
*   No sound: Check your system volume mixer. Ensure the game process is not muted.
*   Long loading times: This happens because the game loads many files at once. The game will load faster every subsequent time you start it.
